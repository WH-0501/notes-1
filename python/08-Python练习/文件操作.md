# 一个文本有100行内容,现分10行 写到10个文件

```Python
#!/usr/bin/env python3
# _*_ coding: utf-8 _*_

for n in range(0,100,10):
    with open('file') as reader, open(str(n // 10)+".txt", 'w') as writer:
        for index, line in enumerate(reader):
            if index >=n and index < n+10:
                writer.write(line)
```
