## 系统信息

### mac

```shell
system_profiler命令：显示Mac的硬件和软件信息
sw_vers命令：OSX系统版本
uname命令：显示操作系统名
```

### Linux

查看内核版本

```shell
uname -a

cat /proc/version
```

查看Linux版本

```shell
lsb_release -a

/etc/issue
/etc/system-release
/etc/redhat-release

/etc/centos-release

# ubuntu
/etc/os-release
```
